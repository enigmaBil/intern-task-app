# Variables d'environnement Frontend

Ce projet utilise le fichier `.env` à la racine du monorepo.

## Variables utilisées par le frontend :

### NextAuth
- `NEXTAUTH_URL` : URL de l'application frontend
- `NEXTAUTH_SECRET` (ou `AUTH_SECRET`) : Secret pour les tokens JWT

### API Backend
- `NEXT_PUBLIC_API_URL` : URL de l'API backend (avec `/api/v1`)
- `NEXT_PUBLIC_BACKEND_URL` : URL du backend

### Keycloak (si utilisé)
- `NEXT_PUBLIC_KEYCLOAK_ISSUER` : URL de l'issuer Keycloak
- `KEYCLOAK_ISSUER` : URL interne Keycloak

## Configuration

Toutes les variables sont définies dans `/mini-jira/.env` à la racine du projet.

Les variables préfixées par `NEXT_PUBLIC_` sont accessibles côté client (navigateur).
Les autres variables sont uniquement accessibles côté serveur (API routes, middleware).
